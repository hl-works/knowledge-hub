Config Sendcloud × GLS pour Haute Fidélité Shopify

Claude · HL · 3 avril 2026 · Haute Fidélité
haute-fidelitelogistiqueen-cours

Contexte

Je voulais mutualiser la logistique e-commerce de Haute Fidélité avec l'infrastructure déjà en place chez Cobra (compte Sendcloud partagé + passage GLS quotidien), tout en gardant un suivi client distinct aux couleurs HF.

Ce qui a été construit

  • Confirmation que le compte Sendcloud multi-boutiques est jouable (HF Shopify déjà connectée)
  • Transporteur arrêté : GLS (pickup quotidien existant), colis 300–600g
  • Décision : pas d'assurance complémentaire, couverture de base GLS suffisante
  • Début de config : page Boutiques connectées, étape suivante = branding suivi

Ce qui était difficile / inattendu

  • GLS ne livre pas en BAL — option recadrée rapidement
  • Le branding suivi n'est pas dans Réglages mais dans un menu dédié Suivi/Tracking

Stack

Sendcloud, GLS, Shopify.

Ce que ça illustre

Session de config opérationnelle guidée en temps réel, avec screenshots — Claude lit l'interface, recadre les fausses pistes (BAL/GLS) et séquence les étapes sans surcharger.

Reste à faire

  • Localiser et configurer le branding suivi HF dans Sendcloud
  • Créer la règle d'expédition GLS automatique sur la boutique HF
  • Vérifier que le contrat GLS Cobra est accessible depuis la boutique HF
