Cobra · Pipeline Claude → Claude Code

Agent de création automatique de produits Odoo

Claude · Cobra · mai 2026 · Cobra
agentdataen-cours

Contexte

Cobra créait manuellement chaque nouveau produit Odoo depuis des catalogues dispersés sur plusieurs supports (emails, Excel PDV, photos de tableaux de prix). Objectif : automatiser entièrement ce workflow pour n'importe quelle marque.

Ce qui a été construit

  • Triangulation multi-sources sur catalogue Sony Bravia FY26 : prix d'achat depuis Excel PDV, croisement avec screenshot email (gamme, taille, EAN), prix de vente TTC depuis photo
  • Excel structuré Drive-ready : Nom formaté, Référence interne, EAN, Code fournisseur, Prix achat HT, Prix vente TTC — code couleur par gamme
  • Claude Code a pris le relais : update lst_price via XML-RPC

Ce qui était difficile

  • EAN stockés en float scientifique dans Excel (4.548736e+12) — cast int obligatoire
  • Données sur 3 supports sans format commun : matching visuel/manuel — le vrai goulot à résoudre dans la version générique

Stack

Python, pandas, openpyxl, XML-RPC Odoo, Claude Code.

Ce que ça illustre

Triangulation multi-sources et pipeline Claude → Claude Code : Claude structure les données depuis des sources hétérogènes, Claude Code exécute les écritures Odoo. Le projet Sony est le premier pattern d'un agent générique pour tout le catalogue Cobra.
