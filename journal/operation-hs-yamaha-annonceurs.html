HiFi Lovers · Marketing × e-commerce

Opération HS Yamaha offert aux annonceurs — Brevo × Shopify

Claude · HL · 11 mai 2026 · HiFi Lovers
HiFi Loversshopifyemaildataen-prod

Contexte

Pour le lancement du premier hors-série HiFi Lovers (70 ans Yamaha), je voulais offrir un exemplaire à mes annonceurs et partenaires HF — avec une mécanique qui restreint les codes aux seuls destinataires ciblés (pas de fuite grand public). Besoin : connecter base annonceurs Excel → Brevo → Shopify avec segmentation et codes promo restrictifs combinables.

Ce qui a été construit

  • Import Brevo de 201 annonceurs depuis l'Excel master HF (attributs catégoriels + opt-in B2B). Liste « Annonceurs HF #16 ».
  • Import Shopify de 143 clients ciblés avec tag annonceur-yamaha-2026 → segment automatique « Annonceurs HS Yamaha 2026 » (142 clients)
  • 2 codes promo Shopify combinables, restreints au segment ET au produit : HIFI-LVRS-1-PART (100% off sur le HS) + HIFI-LVRS-1-PORT (port offert France). Validité 11 mai → 30 juin, 1 usage/client.
  • Template email Brevo en YAML (logo HF, couv HS, kicker rouge collector, lettre signée, encadré monospace pour le 2e code, bloc pédagogique 3 étapes)
  • CTA email pré-appliquant le 1er code via URL discount Shopify
  • Campagne « HS Yamaha offert - Annonceurs - Mai 2026 » programmée le 12 mai 10h, 147 destinataires, UTM activé

Ce qui était difficile / inattendu

  • L'éditeur YAML de Brevo est très pointilleux (rejette les #, syntaxe stricte des tirets) → validation Python du YAML obligatoire avant envoi
  • Shopify ne permet pas un seul code combinant produit gratuit + port gratuit → 2 codes combinables (productDiscount + shippingDiscount avec combinesWith). Tradeoff : saisie manuelle du 2e code au checkout, d'où le bloc pédagogique dans l'email.

Stack

Excel master HF (pandas) → Brevo (import contacts, éditeur YAML) → Shopify (segments via tags, discountCodeBasicCreate + discountCodeFreeShippingCreate avec customerSelection par segment, combinesWith) → URL discount Shopify pour pré-application. Polices Syne + Montserrat. Couleurs HiFi Lovers (#d11514 / #292626 / #f5f2ec).

Ce que ça illustre

Orchestration end-to-end multi-outils (Excel → email marketing → e-commerce) avec Claude qui fait le data wrangling, la config marketing, la rédaction éditoriale et le code GraphQL Shopify. Mécanique business non triviale (codes combinables restreints par segment + produit) traduite en config réelle, testée en checkout avant envoi.
