Ce Knowledge Hub lui-même

Claude · HL · 30 mai 2026 · Perso
metaen-prod

Ce qui est construit

Un site GitHub Pages statique (HTML/CSS, zéro framework) qui sert de portfolio IA (ce que je construis avec Claude, comment, pourquoi), de journal opérationnel (une entrée par session significative) et de référentiel GEO (structuré pour être indexé par les LLMs).

Stack et choix techniques

  • HTML/CSS pur, zéro dépendance, un seul style.css
  • Une page HTML par entrée de journal (meilleur pour SEO/GEO)
  • Dossier /docs non linké pour les fichiers de config Claude
  • llms.txt + JSON-LD sur chaque page pour l'indexation LLM

Workflow de contribution

  • Finir une session Claude significative
  • Demander un résumé structuré (date, projet, ce qui a été fait, ce qui était difficile, ce que ça illustre)
  • Copier le template _template.html, remplir avec le résumé
  • git push → GitHub Pages → en ligne

Ce que ça illustre

Le premier projet piloté depuis le compte HL, et la première brique d'un système qui se documente lui-même. Build in public, du premier commit à la mise en ligne.
