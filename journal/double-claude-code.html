Double Claude Code : Cobra vs HL

Claude · HL · 30 mai 2026 · Perso
metaen-prod

Ce qui s'est passé

Pendant longtemps, un seul Claude Code : le compte Cobra, sur le Mac pro, pour le dev Odoo/Shopify.

En mai 2026, création d'un second setup pour les projets HL (personnels + groupe HL) : compte hugo.lahutte@gmail.com · Claude Max · Opus 4.8, alias terminal claude-hl avec clé API dédiée, interface web via claude.ai/code connectée au repo GitHub personnel.

Pourquoi deux comptes

  • Cobra : contexte pro, mémoire des modules Odoo, intégrations Shopify, collègues potentiels
  • HL : projets perso, stratégie, Knowledge Hub, HiFi Lovers, expérimentations

Les deux peuvent contribuer au même repo GitHub via des résumés de session en markdown → push → en ligne.

Ce que ça change

Le Knowledge Hub lui-même est le premier projet piloté depuis le compte HL. Le repo hugolahutte/knowledge-hub est connecté à claude.ai/code.

« J'ai deux Claude. L'un connaît Odoo 18 par cœur et sait ce que fait chaque module de cobrason.odoo.com. L'autre me suit sur mes projets perso et ne parle jamais d'ERP. Les deux peuvent écrire dans le même journal. »
