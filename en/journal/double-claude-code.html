Two Claude Code accounts: Cobra vs HL

Claude · HL · May 30, 2026 · Perso
metaen-prod

What happened

For a long time, there was just one Claude Code account: the Cobra one, on the work Mac, for Odoo/Shopify dev.

In May 2026, I set up a second setup for HL projects (personal + HL group): account hugo.lahutte@gmail.com · Claude Max · Opus 4.8, terminal alias claude-hl with a dedicated API key, web interface via claude.ai/code connected to the personal GitHub repo.

Why two accounts

  • Cobra: work context, Odoo module memory, Shopify integrations, potential colleagues
  • HL: personal projects, strategy, Knowledge Hub, HiFi Lovers, experiments

Both can contribute to the same GitHub repo via structured session summaries in markdown → push → live.

What changes

The Knowledge Hub itself is the first project driven from the HL account. The hugolahutte/knowledge-hub repo is connected to claude.ai/code.

"I have two Claudes. One knows Odoo 18 inside out and knows what every module on cobrason.odoo.com does. The other follows me on my personal projects and never mentions ERP. Both can write to the same journal."
